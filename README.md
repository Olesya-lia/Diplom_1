# Diplom_1

Учебный проект Яндекс Практикум.
Курс "Инженер по тестированию: от новичка до автоматизатора".

Автоматизация тестов для Stellar Burgers

Подключение библиотек: JaCoCo, Mockito, JUnit 4.
Юнит-тесты для класса Burger c использованием мок и параметризации.
100% покрытие.
